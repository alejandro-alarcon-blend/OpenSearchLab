# Implementación de Ejemplo con OpenSearch y OpenSearch Dashboards

## Descripción

Este proyecto corresponde a una implementación de ejemplo que utiliza dos contenedores Docker:

- Un contenedor para **OpenSearch**, que funciona como motor de búsqueda y análisis de datos.
- Un contenedor para **OpenSearch Dashboards**, que proporciona la interfaz gráfica para visualización y consulta.

El objetivo de esta implementación es mostrar, de forma general, cómo ejecutar ambos servicios mediante contenedores Docker.
